Creating/Editing event details

  1. Go to www.wags.org/wagsadmin
  2. Click on "Events"

    either

  3. To add a new fixture go to the bottom of the page and press one of:
    • Add Event - to add a new VL fixture
    • Add Tour - to add a new Tour. (Add the events for the tour first)
    • Add Non Event - to add a new non-VL fixture (e.g. Burhill, Wags social)

    or

  4. To edit an event's details, click on the arrow for "Details" against the event
  5. You can then fill in the details
  6. For the schedule, you can leave unused time slots empty.
  7. Press Save (at the bottom) or press the browser Back button to cancel without saving.
