Add or edit news items

This allows you to create or edit News items for the "What's New" section of the web site. You can have several items under one date.

  1. Go to www.wags.org/wagsadmin
  2. Click on "News"

    either

  3. Choose a date and Click on Edit news item

    or

  4. Click on Add news item
  5. The screen shown next allows you to add several bullet items for a news date together with an optional covering message.
    To add a new bullet, select the item to add from the drop-down at the bottom. There are a number of "standard" news items to choose from:
    • Open booking. For the next event
    • Event Result. For the last event
    • Handicap update. After the last event
    • Account update.
    • Publish minutes.

    Each item will create a link which can be tested using the purple arrow

  6. When finished, press Save (at the bottom) or press the browser Back button to cancel without saving.
