Maintain member details

This allows you to add a new member or to edit the status, contact details and handicap for an existing member.

  1. Go to www.wags.org/wagsadmin
  2. Click on "Members"

    either

  3. Choose a member from the drop-down list and press Edit member

    or

  4. To add a new member, press Add member
  5. You can then fill in the member's details
  6. It is usually only necessary to give a new handicap value for new members but this also allows a handicap to be arbitrarily set to apply from a given date (which can be in the past or future).
  7. Press Save (at the bottom) or press the browser Back button to cancel without saving.

This page also includes a link to a printable application form for proposing a new member.
