Updating event bookings

Prior to an event going "live", this allows you to edit the bookings for the event - Add or remove players, change a guest name or handicap.

It also allows adding all players for a Tour.

It is important to have a complete and correct list of players (with handicaps for guests) for an event in order to enter the scores.

  1. Go to www.wags.org/wagsadmin
  2. Click on "Events"
  3. Click on the arrow for Bookings against the event
  4. The screen shown will show all bookings to date for the event
  5. To add a player or a guest, click on the Add Player button at the bottom of the screen
  6. If a player is no longer going to attend, untick in the Playing? column
  7. To change a guest name or handicap, enter the new value as appropriate
  8. Click on the Save button to save the changes
