Entering Handicap changes after an event

This enables you to enter handicap changes triggered by players performances at an event. The new handicaps will apply from the day after the event date.

It is also possible to set a handicap for a member at an arbitrary date - see help for Members.

  1. Go to www.wags.org/wagsadmin
  2. Click on "Events"
  3. Click on the arrow under "Handicap" for the event This will show a form with all players in the event. Enter the new handicaps where appropriate. When done, click on "Save" at the bottom. If you miss one or make a mistake, repeat.
  4. Press Save (at the bottom) to save the changes or press the browser Back button to cancel without saving.
